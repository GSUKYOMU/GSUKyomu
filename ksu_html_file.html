안내: 본 시스템은 휴강 및 보강 신청 절차를 안내합니다. 자세한 문의는 교무처로 연락주시기 바랍니다.

⚠️ 중요

교수, 강사는 임의로 결강 및 휴강을 할 수 없습니다. 정당한 사유로 휴강해야 할 경우 반드시 교무처장의 사전 승인을 받아야 합니다.

📅 지정보강일 (학기 시작 전 일괄 처리)

교무처에서 학기 시작 전에 헤이영을 통해 지정보강일로 일괄 처리합니다. 별도의 신청이 필요없습니다.

🔄 지정보강일 변경 신청

지정보강일이 아닌 다른 날짜 또는 온라인 보강을 원할 경우:

💻 PC 방법

  1. 1. 헤이영 메뉴 → 휴강/보강 등록 클릭
  2. 2. 취소할 과목 선택
  3. 3. 일괄처리된 지정보강일 선택
  4. 4. 하단의 휴강취소 클릭
  5. 5. 원하는 일자 및 방식으로 새로 신청

📱 모바일 방법

  1. 1. 메뉴 → 휴/보강조회 클릭
  2. 2. 취소할 과목 선택
  3. 3. 하단의 철회 버튼 터치
  4. 4. 원하는 일자 및 방식으로 새로 신청

🖨️ 헤이영 휴보강계획서 출력 및 제출 절차

1

PC 헤이영 → 메뉴 중 "휴강/보강 수업조회" 클릭

2

오른쪽으로 스크롤하여 "휴보강계획서 [보기]" 클릭

3

매 교시별로 출력

4

우측하단에 본인 서명

5

학과장 및 학과사무실에 제출

6

학과장 서명 후 교무처에 제출

1️⃣ 사전 계획에 따른 휴강 (출장, 결근 등)

  • • 제출시기: 휴강 1주일 전
  • • 제출경로: 학부장 경유 → 교무처
  • • 학생공지: 보강 시간 등을 학생에게 공지
  • • 시스템등록: 전자출결시스템에 휴강/보강 계획 입력

2️⃣ 불가피한 사유에 의한 갑작스러운 휴강 (건강 문제 등)

  • • 제출시기: 복귀 즉시
  • • 제출경로: 학부장에게 제출
  • • 시스템등록: 전자출결시스템에 휴강 내역 입력

3️⃣ 결강 처리 기준

  • • 수업 시간 15분 경과 후에도 출강하지 않으면 결강 처리
  • • 결강 시 보강을 반드시 실시해야 함

📊 횟수 제한

  • • 학기당 과목별 최대 2회까지만 가능
  • • 개인사유 2회 이상: 경고
  • • 개인사유 3회 이상: 허용 불가

📅 기간 제한

  • • 보강은 기말시험 이전까지만 가능
  • • 종강 후 보강 불가
  • • 법정공휴일로 인해 15주 미만 시 16주차는 보충기간

⏰ 제출 기한

보강 실시 후 일주일 이내에 제출해야 합니다.

💻 PC 버전 캡쳐 방법

  1. 1. 해당 과목 출석부 입장
  2. 2. 예시로 2-3명 학생의 해당 주차, 날짜 O/X/△ 클릭
  3. 3. "출력상세보기"의 "변경시간"이 나오도록 캡쳐

📱 모바일 버전 캡쳐 방법

  1. 1. 우측하단 전체메뉴 → 출결관리
  2. 2. 강의목록에서 보강 진행 과목 선택
  3. 3. 보강 진행한 해당 주차 터치
  4. 4. 예시 학생의 출석 옆 ">" 터치
  5. 5. 2-3명의 해당 보강일자 및 시간 출결 내용 캡쳐
